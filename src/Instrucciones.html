INSTRUCCIONES

¿Estas lista (o) para cifrar o descifrar tu mensaje secreto?


1. Introduce el texto que deseas cifrar o descifrar

2. Elije el numero de desplazamiento, este se elige de acuerdo al metodo de cifrado cesar, esto quiere decir que cada letra del texto original es remplazada por otra del mismo alfabeto, depende el desplazamiento que elijas.

3. Pulsa el botón enter

4.Si eseas cifrar o descifrar de nuevo pulsa el boton limpiar.

5.Pulsa el botón salir cuando deses terminar o instrucciones, si no te quedo algo claro.
